# AI Quest Generator

一个基于 Laravel + OpenAI API 的 AI 游戏任务生成平台。

## 项目介绍

AI Quest Generator 可以帮助游戏开发者快速生成任务、NPC 对话和游戏剧情，提高游戏内容制作效率。

## 功能

- AI 生成游戏任务
- AI 生成 NPC 对话
- AI 生成游戏剧情
- 玩家任务管理
- 历史记录保存

## 技术栈

- Laravel
- PHP
- MySQL
- Bootstrap（或 Vue）
- OpenAI API
